# LangGraph 学习项目

一个渐进式学习 LangGraph 框架的代码示例集合，通过从简单到复杂的实际案例，系统掌握 LangGraph 的核心概念和高级特性。

## 📋 项目概述

- **项目类型**: 教育性代码示例集合
- **目标**: 通过一系列渐进式示例，系统性地学习和掌握 LangGraph 框架
- **技术栈**: Python 3.13+、LangGraph、LangChain、OpenAI
- **包管理**: uv

## 🎯 设计原则

1. **渐进式学习**: 从简单到复杂，逐步引入新概念
2. **一致性**: 所有示例遵循相同的代码结构和规范
3. **可复现性**: 每个示例都能独立运行
4. **文档完整性**: 每个示例都有详细的 README 说明

## 🚀 快速开始

### 环境要求

- Python 3.13+
- uv（包管理器）
- OpenAI API Key（用于示例运行）

### 安装依赖

```bash
# 安装项目依赖
uv sync

# 激活虚拟环境
uv run python --version
```

### 运行示例

```bash
# 进入示例目录
cd example_01

# 运行示例
uv run python article_generator.py

# 或者使用提供的脚本
./run.sh
```

### 配置环境变量

复制环境变量模板并配置：

```bash
cp .env.example .env
# 编辑 .env 文件，添加你的 OpenAI API Key
```

## 📖 学习路径建议

1. **初学者阶段** (1-3 天)

   - 从 Example 01 开始，掌握基本概念
   - 学习 Example 02，理解条件分支
   - 完成 Example 03，掌握循环机制

2. **中级阶段** (4-7 天)

   - 学习 Example 04，掌握工具调用
   - 完成 Example 05，理解自适应系统
   - 学习 Example 06，掌握人在环机制

3. **高级阶段** (8-14 天)

   - 完成 Example 07，理解多 Agent 协作
   - 学习 Example 08，掌握反思机制
   - 完成 Example 09，理解分层规划

4. **专家阶段** (15+ 天)
   - 挑战 Example 10，实现自主学习
   - 尝试扩展和优化现有示例
   - 应用于实际项目场景
